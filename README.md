# Plataforma BIM 3D

Plataforma web para visualização de modelos BIM 3D, gestão de projetos, versionamento de modelos e criação de issues vinculadas ao modelo.

## Objetivo

Criar uma aplicação web para upload, organização e visualização de modelos 3D oriundos do Navisworks, com controle de acesso por usuário.

## Funcionalidades previstas

- Cadastro e login com e-mail e senha
- Criação de projetos
- Organização de modelos por projeto
- Upload de modelos
- Versionamento de arquivos
- Visualização 3D
- Criação e gestão de issues
- Perfis de acesso:
  - Administrador
  - Visualizador com permissão para criar issues

## Stack inicial sugerida

- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js
- PostgreSQL
- Autodesk Platform Services Viewer
